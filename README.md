#项目基础目录结构

**用到的插件**

```
    babel-preset-es2015: babel转码规则，将es6语法转成es5
    gulp-autoprefixer: css自动添加前缀
    gulp-babel: babel转码工具
    gulp-cache: 缓存，用于压缩图片时，只压缩改变过的图片
    gulp-clean: 文件清除
    gulp-clean-css: css压缩
    gulp-concat: 文件合并
    gulp-imagemin: 图片压缩
    gulp-rename: 文件重命名
    gulp-sass: sass预编译
    gulp-str2hex: js hex转码混淆
    gulp-uglify: js压缩
    imagemin-pngquant: imagemin的深度压缩工具
```
